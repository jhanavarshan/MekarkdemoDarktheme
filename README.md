# Mekark – EOT Crane Manufacturer | Dark Theme

A premium **dark-themed** version of the Mekark industrial website.

## 🌑 Dark Theme Features
- Deep dark backgrounds (`#0d0d0d`) with layered glassmorphism panels
- Vivid crimson red (`#FF3333`) accent colour for CTAs, labels & highlights
- High-contrast white/light text on all sections
- Scroll-scrubbed video background with dark overlay
- All micro-animations, 3D card tilts and scroll reveals preserved
